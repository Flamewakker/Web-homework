Список определений

HTML
Язык разметки гипертекста (Hypertext Markup Language) для создания веб-страниц.
CSS
Каскадные таблицы стилей (Cascading Style Sheets) используются для описания внешнего вида документа, написанного на языке HTML.
JavaScript
Язык программирования, используемый для создания интерактивных пользовательских интерфейсов, анимации и других функций веб-страниц.
HTTP
Протокол передачи гипертекста (Hypertext Transfer Protocol), используемый для обмена данными между клиентским и серверным компьютерами в Интернете.
URL
Адрес ресурса в Интернете (Uniform Resource Locator), который позволяет уникально идентифицировать веб-страницу или другой ресурс.
