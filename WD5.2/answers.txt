Класс navbar-expand-lg отвечает за определение, когда навигационное меню должно переходить в расширенный (большой) режим, то есть когда оно должно становиться полноэкранным на устройствах с широким экраном. В данном случае, "lg" означает, что это происходит при ширине экрана, равной или большей 992px.

2. Чтобы добавить разделители во вложенном меню, необходимо использовать класс dropdown-divider. Этот класс применяется к элементу списка (обычно <li>), который является разделителем. Например, чтобы добавить разделитель между двумя пунктами меню, код может выглядеть так:

<ul class="navbar-nav">
  <li class="nav-item">
    <a href="#" class="nav-link">Пункт меню 1</a>
  </li>
  <li class="nav-item dropdown">
    <a href="#" class="nav-link dropdown-toggle" data-toggle="dropdown">Вложенное меню</a>
    <ul class="dropdown-menu">
      <li><a href="#" class="dropdown-item">Подпункт меню 1</a></li>
      <li class="dropdown-divider"></li>
      <li><a href="#" class="dropdown-item">Подпункт меню 2</a></li>
    </ul>
  </li>
  <li class="nav-item">
    <a href="#" class="nav-link">Пункт меню 3</a>
  </li>
</ul>
Здесь dropdown-divider применен к элементу списка <li> между двумя подпунктами во вложенном меню.
